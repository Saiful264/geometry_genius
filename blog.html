The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The purpose is we can use to manipulate the HTML, style, and inner document of a web page. Uses the DOM to access the document and its elements. DOM is not a programming language but without it java script language don’t have those modern feature like the notion of the web page, HTML documents etc.

There are 5 ways in which you can select elements in a DOM using selectors.
  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event.
